# mockups

A simple project for iterating on UI mockups with nested navigation.

## Usage

```bash
npm install
npm start
```

Open http://localhost:3000

## Structure

- `public/` - Home page and shared styles
- `mockups/` - Organized by category (e.g., `forms/international/commercial-invoice`)

## Adding Mockups

1. Create folder under `mockups/` (e.g., `mockups/category/subcategory/name/`)
2. Add `index.html` in the folder
3. Add navigation link in `public/index.html`
